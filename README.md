## Description

Simple CRUD Application with Nest js and Mongo DB.

## Installation

```bash
$ npm install
```

## Start Mongo db

* make sure to create <span style="color:red">*C:\data\db*</span> folder 
* go to the mongodb directory of your computer and run the mongod.exe

```bash
$ mongod.exe
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
